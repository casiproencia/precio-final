# Precio Final - Calculadora de Precios con Impuestos

**App web para calcular automáticamente los precios de electricidad de Octopus con todos los impuestos incluidos (2,5% impuesto eléctrico + 21% IVA).**

---

## ¿Cómo funciona?

1. **Sube una captura de los precios horarios** desde la app de Octopus.
2. El sistema detecta automáticamente los **tramos horarios y precios base**.
3. Calcula el **precio final aplicando el 2,5% + 21% (total: 1,24025)**.
4. Muestra una tabla clara con los datos y un botón para **descargar la imagen final**.

---

## Características

- Estilo visual adaptado a los colores de Octopus.
- Cálculo automático del **precio con impuestos**.
- **Responsive**: diseñado para verse perfectamente en iPhone.
- Botón para **guardar el resultado como imagen JPG**.
